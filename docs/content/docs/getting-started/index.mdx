# Getting started

Node **22–24** is supported (`engines.node: ">=22 <25"`).

## Quick start

From the repository root (not `docs/`):

```bash
npm ci
npm run dev
```

Open `http://localhost:3000`, sign in with email, enter the Instant magic code.
